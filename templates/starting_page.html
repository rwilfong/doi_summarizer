Welcome to DOI Summarizer and Recommender!

Program Overview
This program is designed to summarize and recommend articles based on their DOI (Digital Object Identifier). It utilizes natural language processing techniques to extract key information and generate concise summaries. Additionally, it offers personalized article recommendations based on user input papers using a content-based recommendation system. It recommends papers based on both the keywords and the MeSH terms.

How to Use
To summarize and recommend articles, please select "DOI Summarizer" in the top right. To return previous papers based on table name and date, please select "Return Data" in the top right.
Disclaimer: Not all DOIs will have recommended papers.
